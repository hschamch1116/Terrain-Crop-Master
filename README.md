# Terrain Crop Master

QGIS에서 CryEngine용 지형 제작에 필요한 크롭바운드 생성, DEM/항공사진/일반 이미지 래스터 내보내기, RAW 변환을 한 창에서 처리하는 플러그인입니다.

## 주요 기능

### 크롭바운드 생성 및 관리

- 지도 클릭 한 번으로 2048m, 4096m, 8192m, 15360m 크롭바운드 박스를 생성합니다.
- 크롭바운드 레이어 드롭다운에서 현재 작업 대상 레이어를 선택합니다.
- 선택된 크롭바운드는 강조된 드롭다운과 상태 배지로 표시됩니다.
- 편집 모드 전환과 마우스 이동 기능으로 박스 위치를 조정할 수 있습니다.
- Google Maps 링크 또는 위도/경도 좌표를 입력해 해당 위치로 지도를 이동할 수 있습니다.

### DEM 영역 맞춤 및 8192 이미지 정렬

- **DEM 영역으로 크롭바운드/8192 맞춤** 버튼으로 선택한 DEM 또는 래스터 extent에 크롭바운드 박스를 이동합니다.
- 선택한 8192x8192 이미지/래스터가 있으면 이동된 크롭바운드에 맞는 지오리퍼런스 VRT 레이어를 생성합니다.
- 원본 이미지는 수정하지 않고 `fitted_rasters` 폴더에 VRT만 생성합니다.

### 수동 내보내기

- **수동 내보내기 설정 및 실행** 버튼은 레이어 패널에서 직접 선택한 레이어만 처리합니다.
- 프로젝트 전체를 자동으로 찾아 내보내는 자동 일괄 내보내기 기능은 제거되었습니다.
- DEM은 CryEngine BT, GeoTIFF, Hillshade GeoTIFF로 내보낼 수 있습니다.
- 항공사진 및 일반 이미지 래스터는 GeoTIFF 또는 BMP로 내보낼 수 있습니다.
- 좌표계가 없는 일반 이미지 래스터도 QGIS 렌더링 방식으로 크롭바운드 영역에 맞춰 저장합니다.
- 여러 이미지 래스터를 선택해도 출력 파일명에 레이어 이름을 포함해 덮어쓰기를 줄입니다.

### 저장 기능 통합

- **크롭바운드/선택 레이어 저장** 버튼 하나로 저장 흐름을 통합했습니다.
- 레이어 패널에서 선택한 임시 메모리 레이어가 있으면 해당 레이어를 GeoPackage 또는 Shapefile로 저장합니다.
- 선택한 임시 레이어가 없으면 현재 크롭바운드 박스를 Shapefile로 저장합니다.
- `Ctrl + S`도 같은 통합 저장 동작을 실행합니다.

### RAW 변환

- DEM 결과물을 CryEngine에서 사용할 수 있는 16-bit RAW 파일로 변환합니다.
- 변환 시 Little Endian 정렬, 90도 회전, 불필요한 헤더 제거 흐름을 포함합니다.
- 최소/최대 높이를 기준으로 스케일 정보를 계산해 파일명에 반영합니다.

## 기본 사용 흐름

1. 플러그인 버튼을 눌러 컨트롤러를 엽니다.
2. 지도에서 크롭바운드를 생성하거나 기존 크롭바운드 레이어를 드롭다운에서 선택합니다.
3. 필요하면 DEM/래스터 레이어를 선택하고 **DEM 영역으로 크롭바운드/8192 맞춤**을 실행합니다.
4. 내보낼 DEM, 항공사진, 일반 이미지 래스터, 벡터 레이어를 레이어 패널에서 직접 선택합니다.
5. **수동 내보내기 설정 및 실행**을 눌러 해상도와 출력 포맷을 선택한 뒤 저장합니다.
6. 필요하면 **크롭바운드/선택 레이어 저장**으로 크롭바운드 또는 임시 레이어를 영구 저장합니다.
7. DEM 결과가 필요하면 RAW 변환기를 실행합니다.

## 단축키

| 단축키 | 동작 |
| --- | --- |
| `Ctrl + E` | 선택한 크롭바운드 레이어 편집 모드 전환 |
| `Ctrl + M` | 선택한 크롭바운드 이동 도구 활성화 |
| `Ctrl + S` | 선택 임시 레이어 저장 또는 현재 크롭바운드 SHP 저장 |

## 출력 파일

- DEM BT: `Terrain_<size>m_<pixels>.bt`
- DEM GeoTIFF: `Terrain_<size>m_<pixels>.tif`
- Hillshade GeoTIFF: `Hillshade_<size>m_<pixels>.tif`
- 항공사진/일반 이미지 GeoTIFF: `Satellite_<layer>_<size>m_<pixels>.tif`
- 항공사진/일반 이미지 BMP: `Satellite_<layer>_<size>m_<pixels>.bmp`
- 8192 맞춤 VRT: `fitted_rasters/<layer>_fit_<crop_layer_id>.vrt`

## 변경 로그

### v1.6.0

- 자동 DEM + 항공사진 일괄 내보내기 버튼과 자동 탐색 분기를 제거했습니다.
- 수동 내보내기는 레이어 패널에서 사용자가 직접 선택한 레이어만 처리하도록 정리했습니다.
- 일반 이미지 래스터가 DEM으로 오분류되어 항공사진 내보내기가 실패하던 문제를 수정했습니다.
- 좌표계 없는 일반 이미지 래스터도 QGIS 렌더링 방식으로 GeoTIFF/BMP 내보내기가 가능하도록 수정했습니다.
- DEM 또는 래스터 extent에 맞춰 크롭바운드 박스를 이동하고, 8192x8192 래스터를 같은 크롭바운드에 VRT로 맞추는 기능을 추가했습니다.
- 크롭바운드 레이어 선택 드롭다운과 상태 라벨의 시인성을 강화했습니다.
- 현재 박스 SHP 저장과 선택 임시 레이어 저장을 하나의 저장 버튼으로 통합했습니다.
- 여러 일반 이미지 래스터 내보내기 시 파일명에 레이어 이름을 포함해 덮어쓰기를 줄였습니다.
