# 지형 거리별 크롭 저장기 (Terrain Crop Master)

> **QGIS 기반 CryEngine 지형 제작 올인원 가공 플러그인**
>
> QGIS 지도상에서 한 번의 클릭으로 정교한 정사각형 크롭 영역(크롭 박스)을 생성하고, DEM, 위성/항공사진(래스터), 벡터(Shapefile/GeoPackage) 스타일을 지정된 크기로 정밀 크롭하여 내보내는 지형 제작 특화 파이프라인 툴킷입니다.

---

## ⚡ 핵심 기능 (Key Features)

1. **다중 크롭 박스 일괄 생성 및 타겟 바인딩**
   - 클릭 지점을 기준으로 `2048m`, `4096m`, `8192m`, `15360m` 규격의 정사각형 레이어를 복수 체크하여 **일괄 배치**합니다.
   - 여러 개의 크롭 영역 배치 시 가장 큰 영역이 메인 제어 대상 레이어로 자동 매핑되며, 제어 센터 드롭다운 목록이 실시간 동기화됩니다.

2. **완벽한 스타일 보존 렌더링 (벡터 레이어 래스터화)**
   - QGIS 캔버스에 표현된 선, 면, 점 스타일 및 라벨을 깨짐 없이 고해상도 알파(Alpha) 채널 GeoTIFF로 복사 렌더링합니다.
   - 좌표계 투영 정의를 지오태깅으로 완벽히 주입하여 내보냅니다.

3. **고해상도 다중방향 음영기복도 (Multidirectional Hillshade)**
   - 고도 45도, Z척도 1, 다중 광원 조건이 최적화된 고급 음영기복도(Hillshade)를 원터치로 연산합니다. 

4. **크라이엔진 호환 RAW 변환기 연동**
   - 크롭 저장된 DEM 고도 결과물을 크라이엔진(CryEngine)에 다이렉트 임포트할 수 있도록 16비트 리틀엔디안 UInt16 RAW(.raw) 포맷 변환기를 내장 지원합니다.
   - 변환 저장 시 파일명에 높이 스케일 정보 `(Max - Min) * 2`가 자동 산출되어 파일명 접미사로 기록됩니다.

5. **구글맵 링크 화면 이동 및 좌표 자동 투영**
   - 제어 센터 UI 상단에 구글맵 주소 링크 또는 위경도 좌표를 입력창에 넣고 엔터를 치면, 해당 위치(WGS 84)를 QGIS 현재 투영 좌표계로 실시간 변환하여 지도의 화면 중심으로 자동 정렬 이동(Pan/Center)합니다.

6. **순수 BMP 이미지 단독 추출 옵션**
   - 수동 내보내기 시 위성사진(래스터) 및 벡터 레이어에 대해 공간 좌표 정보가 전혀 들어있지 않은 깨끗한 Windows BMP(.bmp) 단독 이미지 파일로 내보낼 수 있어 텍스처 가공이 용이합니다.

7. **자원 안전성 및 예외 방어 설계 (Defensive Programming)**
   - **C++ 소멸 예외 감지**: 사용자가 QGIS 레이어 패널에서 임시 레이어를 수동 제거하는 상황을 인지해, 메모리에서 소멸된 C++ 포인터 접근으로 인한 `RuntimeError` 및 프로그램 다운(Crash) 현상을 완전히 예방합니다.
   - **GDAL 리소스 락 프리**: 윈도우 환경에서 연산 중이던 DEM 파일 핸들이 잠겨 결과물이 깨지거나 유실되는 문제를 GDAL 인스턴스 명시 해제(Garbage Collection)를 적용해 원천 봉쇄했습니다.

## ⌨️ 단축키 안내 (Shortcuts)

사용자의 원활한 작업을 위해 제어기 UI 활성화 시 다음 단축키를 즉시 매핑하여 지원합니다:

| 기능 | 단축키 | 설명 |
| :--- | :--- | :--- |
| **편집 토글** | `Ctrl + E` | 선택된 크롭 바운더리 레이어의 편집 모드를 즉시 켜고 끕니다. |
| **사각형 이동** | `Ctrl + M` | 사각형을 원하는 좌표로 드래그 앤 드롭 이동시킬 수 있도록 지도 맵툴을 활성화합니다. |
| **임시 레이어 저장** | `Ctrl + S` | 레이어 패널에서 다중 선택된 임시 메모리 레이어들을 영구 저장 및 안전 교체합니다. |

## 🚀 사용법 및 워크플로우 (Workflow Guide)

### 1단계: 크롭 박스 일괄 생성
1. QGIS 상단 플러그인 툴바에서 **지형 거리별 크롭 저장기 아이콘**을 클릭합니다.
2. 맵 툴이 활성화되면 지도 캔버스의 중심 좌표가 될 지점을 클릭합니다.
3. 생성 다이얼로그에서 필요한 지형 규격(2048m / 4096m / 8192m / 15360m)을 체크한 후 확인을 누릅니다.
4. 규격별로 매핑된 고유 색상(초록, 빨강, 파랑, 보라)을 띤 박스들이 동시 생성됩니다.

### 2단계: 크롭 바운더리 미세 조절 및 빠른 이동
- **지형 박스 제어 센터** 창에서 `✏️ 편집 모드 켜기` 버튼을 눌러 점 편집을 하거나, `🎯 마우스로 사각형 이동` 버튼을 눌러 위치를 드래그 이동합니다.
- 특정 지역으로 빠르게 이동하고 싶다면 상단의 `🔗 구글맵 링크로 화면 이동:` 칸에 구글맵 주소 링크 또는 `위도, 경도` 좌표를 입력하고 엔터를 누르면 해당 지역으로 즉시 순간이동합니다.

### 3단계: DEM 및 항공사진 원터치 일괄 추출
1. `🚀 DEM + 항공사진 자동 일괄 내보내기` 버튼을 클릭합니다.
2. 타겟 해상도 프리셋(2048px, 4096px, 8192px 등)을 고릅니다.
3. DEM 추출 방식을 선택합니다 (표준 DEM / 음영기복도).
4. 출력 디렉토리를 선택하면, QGIS 내의 모든 DEM 래스터와 항공사진 래스터를 인식하여 정사각형 영역에 맞춰 일괄 크롭 저장합니다.

### 4단계: CryEngine용 RAW 파일 변환
- DEM 추출 완료 후 연동 팝업이 뜨면 `Yes`를 클릭하거나, 컨트롤러에서 `🎮 크라이엔진용 RAW 변환기`를 직접 열어 크롭 완료된 TIF 파일을 RAW 포맷으로 빠르게 변환하여 엔진에 탑재합니다.

## 📅 변경 로그 (Changelog)

### v1.5.0
- **RAW 포맷 명칭 전환**: 기존의 R16(.r16) 포맷 명칭 및 확장자를 표준 RAW(.raw) 규격으로 전면 마이그레이션했습니다. (다이얼로그 타이틀, UI 버튼 라벨, 툴팁, 파일 브라우저 확장자 필터 동기화 완료)
- **자동 높이 스케일 계산 접미사**: RAW 고도 파일 저장 시 지형의 최대 높이에서 최저 높이를 뺀 값의 2배(`(Max - Min) * 2`)를 크라이엔진용 높이 스케일 척도로 자동 연산하여 파일명 뒤에 자동으로 부착하는 기능을 탑재했습니다.
- **순수 BMP 단독 내보내기 지원**: 수동 내보내기 시 위성사진 및 벡터 레이어를 공간 좌표 정보 파일 없이 오직 이미지 픽셀만 깨끗하게 담은 `.bmp` 파일로 내보낼 수 있는 포맷 선택기를 지원합니다. (위성사진 단일 레이어 내보내기 시 DEM으로 오인되던 문제 해결 및 QGIS 캔버스 자동 재로드 제외 처리로 쾌적함 개선)
- **구글맵 위치 추적 네비게이션**: 지형 박스 제어 센터 상단에 구글맵 위치 링크 또는 위경도 좌표값을 붙여넣고 엔터를 치면, 해당 위치를 현재 지도의 좌표계로 투영 변환하여 지도 캔버스 중심으로 자동 정렬 이동합니다.

### v1.4.0
- **가상 메모리 연산 도입**: GDAL 가상 메모리 파일 시스템(`/vsimem/`) 패턴을 적용하여, 디스크 물리적 I/O 락 및 임시 파일 생성 크래시(`Attempt to create new tiff file...`) 현상을 완전히 종식시켰습니다.
- **방위각 옵션 충돌 방지**: 다중방향(`multidirectional`) 활성화 시 방위각(`azimuth`) 매개변수가 C++ 엔진 단에 동시에 중복 전달되지 않도록 동적 매개변수 분기사전(`**opts_dict`)을 설계 및 이식했습니다.
- **좌표계 및 왜곡 수정**: 맵 캔버스 좌표계(`dstSRS`) 재투영 강제 동기화와 축척 환산 수식 보정을 연동하여 수직 사면 왜곡 현상을 해결했습니다.
- **반응형 인터페이스**: 다중방향 모드 활성화 시 무의미한 단일 방위각 슬라이더/인풋 박스를 화면에서 자동으로 Hide 시키는 동적 UI를 이식했습니다.

### v1.3.0
- **다중 크롭 박스 배치**: 2048m, 4096m, 8192m, 15360m 중 필요한 크기를 동시에 한 번의 클릭으로 생성 가능.
- **자동 바인딩 및 동기화**: 일괄 생성 시 최대 규격 레이어로 자동 제어 타겟팅 연계 및 드롭다운 실시간 갱신.
- **QSS 테마 안정화**: UI 전용 스타일시트 통합 및 개별 버튼의 마우스 호버(Hover) 피드백 효과 전격 복원.
- **안정성 강화**: QgsVectorLayer C++ 객체 삭제 감지 처리 추가 (`RuntimeError` 크래시 복구).

### v1.2.0

- Windows 환경의 GDAL 파일 핸들 락(Lock) 해제 처리로 내보내기 신뢰성 제고.
