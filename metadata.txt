about=Terrain Crop Master는 CryEngine 지형 제작을 위한 QGIS 플러그인입니다. 지도 클릭으로 미터 단위 크롭바운드를 생성하고, 선택한 DEM/래스터 영역에 크롭바운드를 맞추며, 8192x8192 이미지 래스터를 지오리퍼런스 VRT로 정렬하고, 사용자가 직접 선택한 DEM/항공사진/일반 이미지 래스터/벡터 레이어를 크롭바운드 기준으로 내보냅니다. DEM 결과물은 RAW 변환기로 CryEngine용 RAW 파일로 변환할 수 있습니다.<br><br><b>주요 기능</b><br>1. 2048m, 4096m, 8192m, 15360m 크롭바운드 생성<br>2. 현재 선택된 크롭바운드 레이어를 드롭다운과 상태 배지로 강조 표시<br>3. 선택한 DEM 또는 래스터 extent에 크롭바운드 박스 이동<br>4. 8192x8192 이미지/래스터를 크롭바운드에 맞춘 VRT 레이어로 추가<br>5. QGIS 레이어 패널에서 직접 선택한 레이어만 수동 내보내기<br>6. DEM을 CryEngine BT, GeoTIFF, Hillshade GeoTIFF로 내보내기<br>7. 항공사진 및 일반 이미지 래스터를 GeoTIFF 또는 BMP로 내보내기<br>8. 좌표계 없는 일반 이미지 래스터도 QGIS 렌더링 방식으로 저장<br>9. 선택한 임시 메모리 레이어 저장과 현재 크롭바운드 SHP 저장을 하나의 버튼으로 처리<br>10. DEM 결과물을 CryEngine용 RAW 파일로 변환
changelog=<b>[v1.6.0]</b><br>- DEM + 항공사진 자동 일괄 내보내기 버튼과 프로젝트 전체 자동 탐색 분기를 제거했습니다.<br>- 수동 내보내기는 QGIS 레이어 패널에서 사용자가 직접 선택한 레이어만 처리하도록 정리했습니다.<br>- 일반 이미지 래스터가 DEM으로 잘못 분류되어 항공사진/이미지 내보내기가 실패하던 문제를 수정했습니다.<br>- 좌표계가 없는 일반 이미지 래스터도 QGIS 렌더링 방식으로 GeoTIFF/BMP 내보내기가 가능하도록 수정했습니다.<br>- 선택한 DEM 또는 래스터 extent에 맞춰 크롭바운드 박스를 이동하는 기능을 추가했습니다.<br>- 8192x8192 이미지/래스터를 이동된 크롭바운드에 맞춘 지오리퍼런스 VRT 레이어로 추가하는 기능을 추가했습니다.<br>- 크롭바운드 레이어 선택 드롭다운과 상태 라벨의 시인성을 강화했습니다.<br>- 현재 박스 SHP 저장 버튼과 선택한 임시 레이어 저장 버튼을 하나의 통합 저장 버튼으로 합쳤습니다.<br>- 항공사진/일반 이미지 래스터 출력 파일명에 원본 레이어 이름을 포함해 여러 레이어 내보내기 시 덮어쓰기를 줄였습니다.<br><br><b>[v1.5.0]</b><br>- R16(.r16) 명칭과 UI를 RAW(.raw) 중심으로 정리했습니다.<br>- RAW 변환 시 높이 스케일 계산과 파일명 기록 흐름을 개선했습니다.<br>- 항공사진/벡터 레이어를 BMP 단독 이미지로 내보내는 옵션을 추가했습니다.<br>- Google Maps 링크 또는 위도/경도 좌표로 지도 화면을 이동하는 기능을 추가했습니다.<br><br><b>[v1.4.0]</b><br>- Multidirectional Hillshade 내보내기 안정성을 개선했습니다.<br>- GDAL 처리 중 임시 파일과 파일 핸들 잠금 문제를 줄였습니다.<br>- 좌표계와 축척 처리 안정성을 개선했습니다.<br><br><b>[v1.3.0]</b><br>- 2048m, 4096m, 8192m, 15360m 다중 크롭바운드 생성 기능을 추가했습니다.<br>- 컨트롤러 UI와 버튼 스타일을 정리했습니다.<br>- 삭제된 QGIS 레이어 참조로 인한 RuntimeError 방어 처리를 강화했습니다.<br><br><b>[v1.2.0]</b><br>- Hillshade GeoTIFF 자동 계산 및 내보내기 기능을 추가했습니다.<br>- Windows 환경의 GDAL 파일 잠금 안정성을 개선했습니다.<br><br><b>[v1.1.0]</b><br>- DEM 출력 포맷 선택 기능을 추가했습니다.<br>- 벡터 레이어 렌더링과 GeoTIFF 좌표 주입 기능을 추가했습니다.<br>- RAW/R16 변환기 연동을 추가했습니다.<br><br><b>[v1.0.0]</b><br>- 거리 기반 사각 크롭바운드 생성과 기본 내보내기 기능을 추가했습니다.
